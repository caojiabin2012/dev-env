# 项目概述

`dev-env` 是一个基于 Tauri 2 + React 19 + TypeScript 的桌面本地开发环境管理工具，当前主路径聚焦于 Web 服务、运行时和中间件的安装、启用、启动、站点绑定与环境配置。

**当前版本**：v0.0.3

## 当前定位

- 主入口是 `src/App.tsx`，默认渲染 `StackPanel`
- 当前 UI 主壳层不再进入旧的工具箱首页
- 仓库中仍保留 JSON、剪切板、日历、计算器等历史模块代码与文档
- 这些旧模块目前更多作为保留能力存在，而不是当前默认产品入口

## 技术栈

- **后端**：Rust + Tauri 2
- **前端**：React 19 + TypeScript + Tailwind CSS v4 + Vite 8
- **状态存储**：JSON 单文件为主；SQLite 主要用于历史剪切板等旧功能
- **系统能力**：托盘、全局快捷键、自动更新、标题栏主题同步
- **发布**：GitHub Actions + GitHub Releases + `latest.json`

## 当前主功能

| 模块 | 说明 |
|------|------|
| 仪表盘 | 展示系统资源、服务状态、常用操作 |
| 软件包 | 下载、安装、启用、启动/停止 Nginx / PHP / MySQL / Redis / RabbitMQ / RocketMQ 等 |
| 站点 | 管理本地站点、目录、域名、Web 服务器和运行时绑定 |
| 环境设置 | 环境根目录、`www` 目录、开机自启、PATH、配置重建 |
| 数据库 | 建库、导入、导出、表级浏览 |
| 运行时 | PHP、Java、Node.js、Python、Go、Composer、pip、npm |

## 项目结构

```text
dev-env/
├── src/
│   ├── App.tsx                   # 当前默认进入 StackPanel
│   ├── components/
│   │   ├── stack/                # 本地环境管理 UI
│   │   ├── settings.tsx          # 设置 / 关于
│   │   └── ...                   # 保留的旧工具模块
│   └── lib/                      # API、Hook、工具函数
├── src-tauri/src/
│   ├── stack/                    # 环境管理核心逻辑
│   ├── settings.rs               # 设置与应用更新安装
│   ├── diagnostics.rs            # 崩溃与前端错误记录
│   └── window_chrome.rs          # Windows 标题栏主题同步
├── release-notes/                # 按版本维护发布说明
└── doc/                          # 开发文档
```

## 运行与交互

- 全局快捷键：`Ctrl+Shift+V`（可配置）
- 托盘行为：左键切换主窗口显示，关闭按钮默认最小化到托盘
- 标题栏主题：Windows 下会跟随应用明暗主题同步浅灰 / 深灰标题栏

## 数据与日志

- 应用数据目录：`%LOCALAPPDATA%\dev-env\`
- 设置文件：`settings.json`
- 崩溃日志：`crash.log`
- 运行日志：`logs/`
